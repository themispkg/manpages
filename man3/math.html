NAME

math.sh - mathematic utils library written in bash5.17

 

SYNOPSIS

source "/usr/local/lib/bash5.17/math.sh"

 

DESCRIPTION

In this library, some mathematical constants are given as variables, and also operations used in mathematics, controllers..

 

VARIABLES


    PI  - Pi
    E   - Naiper's constant
    PC  - Pisagor constant
    TC  - Theodorus constant
    EMC - Euler Mascheroni constant
    GM  - Golden mean
    ETC - Embree Trefethen constant

 

FUNCTIONS

math:sqrt "integer value" - this function need first argv, this parametre just could be integer. It will be calculate square root of the parameter. math:is_integer "value" - this function checks if the value given as a parameter is a number.

 

BUGS

No known bugs.
